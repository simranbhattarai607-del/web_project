 # Mini JavaScript Projects

This repository contains two beginner-friendly web development projects made using HTML, CSS, and JavaScript.

# Projects Included

## 1. Student Budget Manager

A simple budget tracking app for students to manage income and expenses.

# Features
- Add income and expense entries
- Automatic balance calculation
- Delete transactions
- Clean and modern user interface

# Use
Helps students track pocket money, spending, and savings.

---

### 2. Rock Paper Scissors

A fun game where users play against the computer.

# Features
- Play against computer
- Random computer moves
- Live score update
- Attractive responsive design

# Use
Entertainment project to practice JavaScript logic.

---

# Technologies Used
- HTML5
- CSS3
- JavaScript

---

# How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Run the HTML files in any browser.

These projects were created for practice and college assignments to improve front-end development skills.
